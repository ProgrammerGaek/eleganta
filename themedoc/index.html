Folder Structure - top

There are three folders

  • img -> This folder contains the images used in this website
  • js -> This folder contains the javascript or jquery plugins used in this website
  • css -> This folder contains the stylesheets such as css or less used in this website
    • css/styles.css       -> The theme design is in this folder. such as color of text, font size, etc
    • css/bootstrap.css -> Bootstrap framework css is in this file
    • css/styles.less      -> The less version of the theme design

CSS Files and Structure - top

styles.less

If you need to modify the theme less variables. You may do it in this file. Since it is less, majority of the theme design can be configured by modifying the less variables. This makes it extremely easy. You need to use a less compiler. If you do not not LESS, please modify the styles.css file only

Variable Name Description
@footer-background-image Background image for the footer
@footer-call-to-action-background Background image for the call to action
@footer-text-color The text colour of the footer
@footer-copyright-background-colour Background color for the copyright section of the footer
@footer-copyright-text-colour Text colour of the copyright section
@mask-color The background colour when hover on portfolio
@navigation-background-colour The background colour of the navigation
@header-h1-font-size Font size of the h1 element - Page titles use h1 for seo purpose
@p-font-size Font size of the p {paragraph} element

Messages

In-order to display messages, you would need to use bootstrap alert feature

You would need to add the class {alert} and the class responding the type of the message such as

  • To display successful messages you need to add the classalert-success example: class="alert alert-sucess"
  • To display error messages, add the class alert-error example:
  • To display an information to the user, add the class alert-info example: class="alert alert-info"
  • To display warning messages, add the class alert-warning example: class="alert alert-warning"

Carousel - top

On homepage the picture under "Let is shine" is an example of carousel
If you hover your mouse on the image, you would see a overlay appearing. The direction in which the mask appears can be controlled.

  • To appear from left, you need to add the class overylay-move-from-left
  • To appear from right, you need to add the class overylay-move-from-right
  • To appear from top, you need to add the class overylay-move-from-top
  • To appear from bottom, you need to add the class overylay-move-from-bottom
